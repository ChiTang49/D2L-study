# D2L Study

这是一个用于学习《动手学深度学习》的个人笔记仓库。

本仓库主要记录我在学习深度学习过程中的代码练习、概念理解、实验笔记和阶段性总结。学习内容以《动手学深度学习》为主，并结合 PyTorch 进行实践。

## 项目背景

深度学习涉及数学基础、机器学习思想、神经网络结构以及工程实践等多个方面。为了更系统地理解相关知识，本仓库用于保存学习过程中的笔记和实验代码，方便后续复习、整理和扩展。
